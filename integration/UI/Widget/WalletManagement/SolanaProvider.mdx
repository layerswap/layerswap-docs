### Install Solana wallet package

To get started, install the latest version of Solana package.

<CodeGroup>
  ```typescript yarn theme={"system"}
  yarn add @layerswap/wallet-svm
  ```

  ```typescript pnpm theme={"system"}
  pnpm add @layerswap/wallet-svm
  ```

  ```typescript npm theme={"system"}
  npm install @layerswap/wallet-svm
  ```
</CodeGroup>

### Basic example

The example below shows how to preconfigure a basic wallet management with native widget package.

```typescript  theme={"system"}
import { LayerswapProvider, Swap } from "@layerswap/widget";
import { SVMProvider } from "@layerswap/wallet-svm"

export const WidgetPage = () => {
  return (
    <LayerswapProvider 
      config={config}
      walletProviders={[SVMProvider]}
    >
      <Swap/>
    </LayerswapProvider>
  );
};
```
