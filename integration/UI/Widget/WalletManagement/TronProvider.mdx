### Install Tron wallet package

To get started, install the latest version of Tron package.

<CodeGroup>
  ```typescript yarn theme={"system"}
  yarn add @layerswap/wallet-tron
  ```

  ```typescript pnpm theme={"system"}
  pnpm add @layerswap/wallet-tron
  ```

  ```typescript npm theme={"system"}
  npm install @layerswap/wallet-tron
  ```
</CodeGroup>

### Basic example

The example below shows how to preconfigure a basic wallet management with native widget package.

```typescript  theme={"system"}
import { LayerswapProvider, Swap } from "@layerswap/widget";
import { TronProvider } from "@layerswap/wallet-tron"

export const WidgetPage = () => {
  return (
    <LayerswapProvider 
      config={config}
      walletProviders={[TronProvider]}
    >
      <Swap/>
    </LayerswapProvider>
  );
};
```
