The Layerswap V0 API serves as the foundation for the future V8 protocol, which represents the cutting-edge in permissionless & trustless bridging technology. While V0 has already processed billions of dollars in transaction volume, showcasing its robustness and safety, the upcoming V8 protocol will set new standards in security and decentralization.

Layerswap V0 leverages proven mechanisms to ensure secure and reliable transactions, protecting both user funds and data integrity. However, the V8 protocol introduces advanced security enhancements, including trustless operations and atomic swaps, delivering the ultimate assurance for your bridging needs.
